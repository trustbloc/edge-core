# edge-core

Edge Core implements core components that are shared between various edge services. 

## Prerequisites (General)
- Go 1.15

## Targets
```
# run checks and unit tests
make all

# run linter checks
make lint

# run license checks
make license

# run unit tests
make unit-test
```
